## üé® Theme Configuration

### Light Theme Variables
```css
:root {
    --primary-color: #E52E2D;
    --text-color: #454545;
    --bg-color: #fff;
    --card-bg: #fff;
    --border-color: #a3a3a3;
```

### Dark Theme Variables
```css
[data-theme="dark"] {
    --primary-color: #ff4444;
    --text-color: #fff;
    --bg-color: #121212;
    --card-bg: #1e1e1e;
    --border-color: #333;
}
```
